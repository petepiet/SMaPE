# SMaPE Roadmap

## Planned features

### Learn from user calibration choices
When the user makes calibration decisions (which notes to mark, which key colors to assign
in Synthesia mode), record those choices per channel. Over time, use the accumulated data
to suggest or auto-apply the same settings for new videos from the same channel, reducing
the number of clicks needed in future batch runs.

**Status**: Not started. User wants this eventually but not yet — no preset errors allowed
until the model is confident enough.
